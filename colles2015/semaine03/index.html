Généralités sur les sommes/séries et les intégrales

Sommes/séries classiques

Sommes

\(\displaystyle \sum_{k = 0}^n k^0 = n + 1\),\(\displaystyle \sum_{k = 0}^n k^1 = \frac{n (n + 1)}{2}\), \(\displaystyle \sum_{k = 0}^n k^2 = \frac{n (n + 1) (2n + 1)}{6}\),

Sommes géométriques

\(\displaystyle \forall q \neq 1, \ \sum_{k= 0}^n q^k = \frac{1 - q^{n + 1 }}{1- q} \), et si \( \displaystyle |q|<1 , \ \sum_{k= 0}^\infty q^k = \frac{1 }{1- q} \).

Loi géométrique

  • Modélisation du rang d’apparition du premier succès à la répétition de \({\mathcal B}(p). \)

  • Pour \(X \hookrightarrow {\mathcal G}(p) \), on a \(X (\Omega) = {\mathbb{N}}\backslash \{0\}\), et \(\forall k {\geqslant}1, \ {\mathbb P}(X = k ) = p q^{k - 1 }\),

  • Interprétation du reste \(\displaystyle \sum_{k = N + 1 }^{+ \infty} {\mathbb P}(X = k) = q^{N}\) comme fonction d’anti-répartition \({\mathbb P}(X > N )\).

Manipulation de sommmes/séries

Pratique du changement d’indice

Dans \(\sum_{k = 0}^N u_{k + 1}\), on pose \(i = k + 1\), et on substitue dans le t.g. et les bornes.

Sommation télescopique

  • Formule \(\displaystyle \sum_{k = n}^p (u_{k + 1 } - u_k) = u_{p + 1 } - u_n\).

  • Exemples d’application de la décomposition en éléments simples.

Séries à termes positifs

: les sommes partielles \(\nearrow\), donc convergent ssi elles sont majorées.

Intégration

Cadre théorique cette semaine : On intégre une fonction continue sur un segment, puis passage à la limite aux bornes.

Propriétés générales

Linéarité, Chasles, positivité.

Intégrales et primitives

\(\displaystyle \int_a^b f (t ) dt = F (b) - F (a)\) si \(F\) est \( {\mathcal C}^1 \) sur \([a , b]\) et \(F'= f\).

Primitives usuelles

  • Fonctions puissances : pour \(a \neq -1\) : \(\displaystyle \int x^a dx = \frac{1}{a + 1} x^{a + 1}\), et \(\displaystyle \int \frac{dx}{x} = \ln (x)\).

  • Exponentielles : pour \(a \neq 0 \) : \(\displaystyle \int \operatorname{e}^{ax} dx = \frac{1}{a } \operatorname{e}^{ax}\).

Intégration par parties

pour \(u, v\) de classe \({\mathcal C}^1 \) : \(\displaystyle \int_a^b u'v = \left [uv \right ]_a^b - \int_a^b uv' \)

  • Exemple de \(\displaystyle \int_1^{x} \ln (t ) \, dt = x \ln (x ) + 1 - x \), pour \(x > 0\).

Pratique du changement de variables

sur un segment

  • Hypothèses \({\varphi}: [a , b] \to {\mathbb{R}}\) de classe \({\mathcal C}^1\), \(f\) continue sur \({\varphi}([a , b])\).

  • Formule \(\displaystyle \int_a^b f ({\varphi}(t)) \, {\varphi}'(t) \, dt = \int_{{\varphi}(a)}^{{\varphi}(b)} f (x ) \, dx \)

  • Notation : on a posé \(x = {\varphi}(t)\) et \(dx = {\varphi}'(t) \, dt\). Alors \(t=a \leadsto x = {\varphi}(a)\) …
