Théorème du point fixe

Pour \((u_n)\) définie par \(u_{n + 1 } = f(u_n)\), avec \(f\) continue, \((u_n)\) converge, c’est vers un point fixe de \(f\) : \(f (\ell) = \ell\). Étude graphique.
